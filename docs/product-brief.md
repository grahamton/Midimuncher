# Product brief

Problem
- Creative block with hardware.
- Want a laptop "MIDI playground" that keeps the OXI One central.

User story
- Connect OXI One to Windows laptop by USB.
- App generates and routes MIDI back to OXI.
- OXI forwards to downstream synths via MIDI out.

Non goals
- Audio engine.
- DAW replacement.
- Browser-only support in v1.
